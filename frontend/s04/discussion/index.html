HTML Images

An image can be shown using <img> tag along with its src attribute.

<img src="http://placekitten.com/450/450">

<img src="./images/zuitt-logo.png">

Valid image extensions are the following:

  1. png
  2. jpg
  3. gif
  4. svg
  5. webp

Take a look at w3schools attribute references here.
